本篇题目：连续子串

题目

给你两个字符串t和p,要求从t中找到一个和p相同的连续子串,并输出该子串第一个字符的下标

输入

输入文件包括两行 分别表示字符串 t 和 p
保证t的长度不小于p
且t的长度不超过1000000
p的长度不超过10000

输出

如果能从t中找到一个和p相等的连续子串,
则输出该子串第一个字符在t中的下标
下标从左到右依次为1,2,3,...
如果不能则输出 No
如果含有多个这样的子串
则输出第一个字符下标最小的

示例一

输入

AVERDXIVYERDIAN
RDXI

输出

4

代码编写逻辑

本题的目标是 KMP 字符串匹配算法。该算法的主要思想是在匹配过程中，当出现不匹配的情况时，不需要回溯到匹配的起始位置，而是利用已经匹配的信息，尽可能地减少匹配次数。

该算法的实现主要分为两个部分：计算模式串的最长公共前后缀（LPS）和匹配过程。在计算 LPS 的过程中，我们需要维护两个指针 i 和 len，其中 i 从 1 开始遍历模式串，len 表示当前已经匹配的最长前后缀的长度。当 p[i] == p[len] 时，我们可以将 len 加 1，并将 lps[i] 赋值为 len。否则，我们需要将 len 更新为 lps[len - 1]，并继续比较 p[i] 和 p[len]。

在匹配过程中，我们需要维护两个指针 i 和 j，其中 i 从 0 开始遍历文本串，j 从 0 开始遍历模式串。当 t[i] == p[j] 时，我们可以将 i 和 j 分别加 1。当 j == p.length() 时，说明已经匹配成功，此时我们可以输出匹配的起始位置 i - j + 1。否则，当 t[i] != p[j] 时，我们需要根据已经匹配的信息更新 j 的值，具体来说，如果 j != 0，则将 j 更新为 lps[j - 1]，否则将 i 加 1。

下述代码的输出结果为：匹配成功时输出匹配的起始位置，否则输出 “No”。

Code

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        try (Scanner s = new Scanner(System.in)) {
            String t = s.nextLine();
            String p = s.nextLine();
            solveMethod(t, p);
        }

    }

    private static void solveMethod(String t, String p) {
        int[] lps = computeLPS(p);
        int i = 0, j = 0;
        while (i < t.length()) {
            if (t.charAt(i) == p.charAt(j)) {
                i++;
                j++;
            }
            if (j == p.length()) {
                System.out.println(i - j + 1);
                return;
            } else if (i < t.length() && t.charAt(i) != p.charAt(j)) {
                if (j != 0) {
                    j = lps[j - 1];
                } else {
                    i++;
                }
            }
        }
        System.out.println("No");
    }

    private static int[] computeLPS(String p) {
        int[] lps = new int[p.length()];
        int len = 0, i = 1;
        while (i < p.length()) {
            if (p.charAt(i) == p.charAt(len)) {
                len++;
                lps[i] = len;
                i++;
            } else {
                if (len != 0) {
                    len = lps[len - 1];
                } else {
                    lps[i] = 0;
                    i++;
                }
            }
        }
        return lps;
    }

}
